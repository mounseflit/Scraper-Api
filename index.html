Ai Crafters Web Scraping API Documentation

Base URL

https://aicrafters-scraper-api.vercel.app

Demonstration

Scrape Endpoint

GET /scrape

Description

Scrapes content from a specified URL based on the requested method.

Query Parameters

url (required)
The target website URL to scrape from.
Example: ?url=https://example.com

type (optional)
The type of content to scrape. Available options:

  • html: Returns the raw HTML of the page
  • images: Returns all image URLs found on the page
  • text: Returns cleaned text content from the page
  • links: Returns all hyperlinks found on the page

CORS Configuration

  • Origin: All origins allowed (*)
  • Methods: GET, POST
  • Max Age: 3600 seconds
  • Credentials: Supported

Response Format

    {
        "message": "Scraping successful!",
        "pageTitle": "Page Title",
        "result": "Scraped content here"
    }

Example Requests

1. Get all image sources from a website:

GET /scrape?url=https://example.com&type=images

Response Format : 

    {
        "message": "Scraping successful!",
        "pageTitle": "Page Title",
        "result": [
            {
            "url": "https://example.com/image1.jpg",
            }
        ]
    }

2. Get all links from a website:

GET /scrape?url=https://example.com&type=links

Response Format : 

    {
        "message": "Scraping successful!",
        "pageTitle": "Page Title",
        "result": [
            {
            "url": "https://example.com/page1",
            "text": "Link text"
            }
        ]
    }

3. Get cleaned text content from a website:

GET /scrape?url=https://example.com&type=text

Response Format : 

    {
        "message": "Scraping successful!",
        "pageTitle": "Page Title",
        "result" : "Cleaned text content from the page"
    }

4. Get raw HTML content:

GET /scrape?url=https://example.com&type=html

Response Format : 

    {
        "message": "Scraping successful!",
        "pageTitle": "Page Title",
        "result" : "Raw HTML content"
    }

Error Responses : 

// 400 Bad Request
{
    "message": "Please provide a valid URL as a query parameter"
}

// 500 Internal Server Error
{
    "message": "Error occurred during scraping",
    "error": "Error details"
}

Root Endpoint

GET /

Returns this documentation page.

Server runs on port 3000 by default (configurable via PORT environment variable)
